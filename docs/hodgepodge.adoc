Low-level API

Transient storage
-----------------

[source,Clojure]
----
(require [hodgepodge.core :refer [local-storage]])

(def val {:bar 42 :timestamp (js/Date.)})
(assoc! local-storage :foo val)
(assert (= val (get local-storage :foo)))

(assoc! local-storage val :foo)
(assert (= :foo (get local-storage val)))
